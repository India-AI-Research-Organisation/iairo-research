This project established a responsive digital learning framework that dynamically adjusts to a learner's evolving understanding using multimodal content analysis. For collaborations, Please reach out to meghac.tiya@gmail.com

## Outcomes

- Implementation of an adaptive question engine leveraging a Hybrid Reinforcement Learning (RL) algorithm.
- Automated pipeline for extracting timestamped, difficulty-rated questions from video transcripts and slide decks.
- Generation of personalized post-lesson summaries using semantic search and large language models.
